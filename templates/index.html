Shoti API

Retrieve random Shawty Videos with metadata.

GET /shoti

Returns a JSON response containing:

  • author: the uploader's name
  • title: the video title or caption
  • video_url: direct download link without watermark

Example Request:

GET https://yourdomain.com/shoti

Key Features

  • Direct download links with no watermark.
  • Retrieve uploader information and captions.
  • Lightweight and fast API response.
  • Made using Python FastAPI
